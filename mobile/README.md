# BookLearner Mobile Apps

This directory will contain the React Native mobile applications for iOS and Android.

## Status

🚧 **Coming Soon** - Mobile apps are planned for future releases.

## Planned Features

- Native camera integration for capturing book covers
- Photo library access
- Share extension support (iOS)
- Share functionality (Android)
- Shared business logic with the web application

## Development

Mobile app development will use React Native to share code between iOS and Android platforms, and will reuse the service layer from the web application for API integrations.

Stay tuned for updates!
